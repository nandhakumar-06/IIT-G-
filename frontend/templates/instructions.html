{% if role_key == 'admin' %}
System Admin Workflow
  • Use Reports to select Department, then Year, then manage Semester test data.
  • Use Counsellor Activity to track completion and export filtered PDF summaries.
  • Use Users to create and maintain Counselor, HoD, DEO and Principal accounts.
  • Use Database tab for backup, restore and controlled data clear operations.
  • Use Session Monitoring and Message Logs for audit and operations visibility.
{% elif role_key == 'principal' %}
Principal Workflow
  • Dashboard provides institution-level completion summary and top performers.
  • Reports is read-only for institution-wide academic visibility across departments.
  • Counsellor Activity helps review completion status by test and section scope.
  • Database allows safe backup/restore supervision with password confirmation.
{% elif role_key in ['hod', 'chief_admin'] %}
HoD Workflow
  • Your panel is scoped to assigned Department and Year only.
  • Use Reports to review semester tests and counselor progress in your scope.
  • Use Counsellor Activity to inspect pending/complete counselors by test.
  • Use Message Logs for communication audit under your scope.
  • HoD does not upload marksheets; uploads are handled by System Admin or DEO.
{% elif role_key == 'deo' %}
DEO Workflow
  • Your panel is scoped to assigned Department and Year only.
  • Use Reports to upload marksheets and maintain test records in your scope.
  • Use Counsellor Activity to monitor completion status by semester/test.
  • Use Message Logs to verify communication activity in your scope.
{% else %}
Counselor Workflow
  • Dashboard shows your available tests for results communication.
  • Reports gives access to uploaded test records under your assignment.
  • Message Logs tracks sent communication entries for your students.
{% endif %}
Safety Notes
  • Always verify Department, Year, Semester and Test before exporting or updating.
  • Use backup before running any clear/restore operation in Database maintenance.
  • Keep account passwords confidential and rotate them periodically.
